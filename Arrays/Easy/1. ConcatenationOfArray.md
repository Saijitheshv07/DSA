# [1929. Concatenation of Array](https://leetcode.com/problems/concatenation-of-array/description/)

- Basic Array Concatenation, More of Array Manipulation learning Problem

### C++ Solution

```cpp
class Solution {
public:
    vector<int> getConcatenation(vector<int>& nums) {
        vector<int> res = nums;
        for(int i=0;i<nums.size();i++){
            res.push_back(nums[i]);
        }
        //res.insert(res.end(),nums.begin(),nums.end());
        return res;
    }
};

```
### Java Solution

```java
class Solution {
    public int[] getConcatenation(int[] nums) {
        int[] result = Arrays.copyOf(nums, nums.length * 2);
        System.arraycopy(nums, 0, result, nums.length, nums.length);
        return result;
    }
}
```

## Java Array Copy — Quick Notes

### 🧠 Core Idea
- Arrays are **objects (references)** in Java  
- Assignment ≠ copy

```java
int[] a = {1,2,3};
int[] b = a; // ❌ same reference
```

---

## ✅ Ways to Copy Arrays

### 1. Arrays.copyOf (most common)
```java
int[] b = Arrays.copyOf(a, a.length);
```
- Creates new array
- Can resize

---

### 2. System.arraycopy (fastest)
```java
System.arraycopy(src, srcPos, dest, destPos, length);
```

Example:
```java
System.arraycopy(a, 0, b, 0, a.length);
```

---

### 3. Arrays.copyOfRange (subarray)
```java
int[] b = Arrays.copyOfRange(a, start, end); // end exclusive
```

---

### 4. Manual loop
```java
for(int i = 0; i < a.length; i++) {
    b[i] = a[i];
}
```

---

## ⚠️ Shallow vs Deep Copy

### Primitives
- Safe → actual values copied

### Objects
```java
Person[] b = Arrays.copyOf(a, a.length);
```
- ❗ Shallow copy (objects shared)

---

## ⚡ When to Use

| Method | Use |
|------|-----|
| copyOf | General ✅ |
| arraycopy | Performance |
| copyOfRange | Subarray |
| loop | Custom logic |

---

## 🚨 Common Mistakes

```java
int[] b = a; // ❌ not a copy
```

```java
Arrays.copyOf(a, smallerSize); // ❌ data loss
```

---

## 🔥 One-liner to remember

```java
int[] copy = Arrays.copyOf(original, original.length);
```
