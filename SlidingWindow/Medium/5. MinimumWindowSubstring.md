

# [76. Minimum Window Substring](https://leetcode.com/problems/minimum-window-substring/description/)


- Variable Size Sliding Window, But faced issue at  ans >= j-i+1, always update ans or make ans INT_MAX at start
- substring should be (start , start + len)
- Ascii values A - 65, a - 97

### Java Solution
```java
class Solution {
    public String minWindow(String s, String t) {
        int[] freq = new int[128];
        int tl = t.length();
        for(int i = 0 ;i < tl; i++){
            freq[t.charAt(i) - 'A']++;
        }

        int i=0;
        int sl = s.length();
        int ans = sl;
        int ct = tl;
        int startIndex = -1;
        for(int j=0;j<sl;j++){
            if(freq[s.charAt(j)-'A'] > 0){
                ct--;
            }
            freq[s.charAt(j)-'A']--;

            while(ct == 0){
                if(ans >= j-i+1){
                    ans = Math.min(ans, j-i+1);
                    startIndex = i;
                }

                if(freq[s.charAt(i)-'A'] >= 0){
                    ct++;
                }
                freq[s.charAt(i)-'A']++;
                i++;
            }
        }
        System.out.println(startIndex);
        if(startIndex == -1) return "";
        return s.substring(startIndex,startIndex+ans);
    }
}
```
