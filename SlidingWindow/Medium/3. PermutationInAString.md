# [567. Permutation in String](https://leetcode.com/problems/permutation-in-string/description/)

- REVISIT
- Here the Key Idea is the window size should be len1 AND ct should be total number of chars in the s1 not just unique chars in s1.

### Java Solution
```java
class Solution {
    public boolean checkInclusion(String s1, String s2) {
        int[] freq = new int[26];
        int ct = 0;
        int len1 = s1.length();
        for(int i=0;i<len1;i++){
            if(++freq[s1.charAt(i)-'a'] > 0) ct++;
        }

        int i=0;
        int len2 = s2.length();
        for(int j=0;j<len2;j++){
            if(--freq[s2.charAt(j)-'a'] >= 0) ct--;

            while(j-i+1 > len1){
                freq[s2.charAt(i)-'a']++;
                if(freq[s2.charAt(i)-'a'] > 0){
                    ct++;
                }
                i++;
            }
            if(ct==0 && j-i+1==len1) return true;
            
        }
        return false;
    }
}
```
